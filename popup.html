ChatGPT API Key Instructions:
1. Sign up for an OpenAI account at https://beta.openai.com/signup/
2. Once you have signed up, go to https://platform.openai.com/account/api-keys to create an API key
3. Follow the instructions to create a new API key. Be sure to select the "Text Completion" option for the API permissions.
4. Copy the API key and save it somewhere secure.

Using ChatGPT with this Plugin:
1. To use your own ChatGPT API key with this plugin, go to the ChatGPT Settings page in your WordPress admin dashboard.
2. Enter your API key in the "API key" field and click "Save All".
